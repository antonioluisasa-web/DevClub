Desafio Git & GitHub

Olá, seja bem-vindo ao desafio de Git & GitHub! Este desafio tem como objetivo testar seus conhecimentos sobre controle de versão e colaboração usando Git e GitHub. Siga as instruções abaixo para completar o desafio.

Instruções

  1. Crie um repositório no GitHub com o nome "desafio-git-github".
  2. Clone o repositório para sua máquina local.
  3. Crie um arquivo chamado "README.md" e adicione uma breve descrição do desafio.
  4. Faça um commit com a mensagem "Adiciona README.md".
  5. Crie uma branch chamada "feature-branch".
  6. Na branch "feature-branch", crie um arquivo chamado "feature.txt" e adicione algum conteúdo relacionado ao desafio.
  7. Faça um commit com a mensagem "Adiciona feature.txt".
  8. Faça um pull request da branch "feature-branch" para a branch principal (main) do repositório.
  9. Aguarde a revisão e aprovação do pull request.

Dicas

  • Certifique-se de ter o Git instalado em sua máquina.
  • Use comandos como `git clone`, `git add`, `git commit`, `git checkout -b`, `git push` e `git pull` para realizar as tarefas.
  • Não se esqueça de escrever mensagens de commit claras e descritivas.

Boa sorte! Estamos ansiosos para ver suas contribuições no repositório!
